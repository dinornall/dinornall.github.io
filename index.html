Plywood Box Layout Simulation

This project involves a MATLAB-based Monte Carlo simulation to calculate the efficiency of cutting plywood sheets into boxes of varying sizes. The program uses a randomized algorithm to place box parts (sides, top, bottom) on the plywood sheet, considering the saw kerf. By running the simulation 1000 times for each box length and analyzing the maximum number of complete boxes and the resulting waste, the project allows a very good estimate of the relationship between box dimensions and material utilization. Results are presented as plots of maximum box yield and waste area versus box length in two separate graphs.
